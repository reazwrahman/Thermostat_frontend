Experimenting With Temperature Sensor (DHT22)

Reaz W. Rahman

Published on June 9, 2024

To evaluate the consistency of the DHT22 temperature sensor's readings, I conducted a series of tests comparing its data to the OpenWeatherMap API. For control, I used the OpenWeatherMap API to retrieve weather data for my current location. Every five minutes, I pinged my backend API to gather readings from the DHT22 sensor and the weather data from OpenWeatherMap. This process continued throughout the day and night. The first set of images shows the data collected during the night of June 4, 2024.

The second set of images, from the day of June 5, 2024, includes plots for temperature and humidity. These plots reveal that the gap between the readings from the DHT22 sensor and the OpenWeatherMap API remained consistent over time. This consistency indicates that the DHT22 sensor provides reliable data. The results demonstrate the sensor's accuracy and reliability, making it a dependable choice for temperature and humidity measurements.
